Privacy Policy

Autodarts Highscore Tracker

The Autodarts Highscore Tracker extension does not collect, transmit, or sell personal data.

Data Processing

This extension processes visible content from the Autodarts website only to detect and display personal highscore-related statistics for the user.

Local Storage

Any stored data is saved locally in the user's browser using the browser storage API. No data is sent to external servers.

Third Parties

No user data is shared with third parties.

Remote Code

This extension does not use remotely hosted code. All executable code is bundled within the extension package.

Contact

If you have questions regarding this privacy policy, please contact the developer through the repository associated with this extension.
